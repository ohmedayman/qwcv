What is QCV?

QCV is a premium, AI-powered CV/resume builder platform competing with tadween.me. It offers 15+ professional ATS-friendly templates, AI scoring, job matching, public portfolio pages with PDF download, and a standalone admin panel. Hosted on GitHub Pages with Firebase backend.

Security & Auth

Authentication

Firebase Auth — Email/password + Google OAuth. Session persistence enabled. Auto-redirect on unauthorized.

Database Security

Firebase RTDB with .validate rules. User data isolated by UID. Admin access via hardcoded credentials.

Admin Panel

Standalone login: admin / Qcv@admin1. 19 sidebar tabs. Audit log tracks all actions. RBAC for staff.

Staff System

Staff login via login.html with Firebase staff/ records. Role-based access control.
